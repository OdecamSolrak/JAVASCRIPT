Tipos de operadores

    Aritméticos:

  1. Soma => +

    2. Ex.: 5 + 2 = 7

  2. Subtração => -

    3. Ex.: 5 - 2 = 3

  3. Multiplicação => *

    4. Ex.: 5 * 2 = 10

  4. Divisão => /

    5. Ex.: 5 / 2 = 2.5

  5. Divisão inteira => %

    6. Ex.: 5 % 2 = 1 (O resultado é o resto da divisão inteira)

  6. Potênciação => **

    7. Ex.: 5 ** 2 = 25 (5² = 25)

Obs.: Ordem de precedência:

  1. Parênteses ()
  2. Potência **
  3. Multiplicação *, Divisão / e Divisão inteira %
  4. Soma + e Subtração -

Atribuição:

É o ato de atribuir um valor a uma variável.

    Exemplo:

  • Atribuição simples é representado pelo sinal de igual (=)

    • var a = 3 + 3 (a = 8)

    var b = a % 5 (b = 3)

    var c = 5 * b ** 2 (c = 45)

  • Auto-Atribuição é quando a atribuição é feita à própria variável, substituindo o valor anterior.

    • Exemplo:

    var a = 3 (a = 3)

    a = a + 4 (a = 7) (simplificação a += 4 (a =7))

    a = a - 5 (a = 2) (simplificação a -= 5 (a = 2))

  • Incremento é quando soma ou subtrai pelo valor (1)

    • Exemplo:

    var a = 5 (a = 5)

    a = a + 1 (a = 6) (simplificação a ++ (a = 6))

    a = a - 1 (a = 5) (simplificação a -- (a = 5))

Relacionais:

Toda a expressão que tenha um operador relacional ligado à ela, o resultado dessa expressão vai ser sempre um resultado booleano (verdadeiro ou falso).

  1. Maior: >

    2. Ex.: 5 > 2 = True (verdadeiro)

  2. Menor: <

    3. Ex.: 7 - 4 = False (falso)

  3. Maior ou igual: >=

    4. Ex.: 8 >= 8 = True (verdadeiro)

  4. Menor ou igual: <=

    5. Ex.: 9 <= 7 = False (false)

  5. Igual: ==

    6. Ex.: 5 == 5 = True (verdadeiro)

  6. Diferente: !=

    7. Ex.: 4 != 4 = False (falso)


  • Relacionais de Identidade

    • Exemplo:

    5 == 5 = True (verdadeiro porque possuem o mesmo valor)

    5 == '5' = True (verdadeiro porque possuem o mesmo valor idependete do tipo)

    5 === '5' = False (falso porque possuem o mesmo valor mais não são do mesmo tipo)

    5 === 5 = True (verdadeiro porque possuem o mesmo valor e são do mesmo tipo)

Lógicos:

  1. Negação => ! (não)

    2. Operador unário, quando só tem um valor lógico (após o operador).

    Após ! entra uma expressão True ou False

    • ! True = O que não é True é False
    • ! False = O que não é False é True
      •

  2. Conjunção => && (e)

    3. Operador binário, quando tem dois valores lógicos (Um antes do operador e um depois)

    • True && True = True
    • True && False = False
    • False && True = False
    • False && False = False
      •

  3. Disjunção => || (ou)

    4. Operador binário, quando tem dois valores lógicos (Um antes do operador e um depois)

    • True || True = True
    • True || False = True
    • False || True = True
    • False || False = False
      •

Exemplo:

var a = 5

var b = 8

a > b && b / 2 == 2 (5 é maoir que 8 e 8/2 = 2) False

a < b || b / 2 == 2 (5 é menor que 8 ou 8/2 = 2) True

Obs.: Ordem de precedência:

  1. Negação => ! (não)
  2. Conjunção => && (e)
  3. Disjunção => || (ou)

Ternário:

Operador Ternário, quando tem três valores lógicos.

? (caso seja True)

: (caso seja False)

Teste lógico ? True : False


Exemplo:

var média = 5.5

média > 7 ? 'Aprovado' : 'Reprvado' (5.5 é maior que 7 caso seja True será Aprovado, caso seja False será Reprovado). O resultado é 'Reprovado'

média += 3

média = 8.5

média > 7 ? 'Aprovado' : 'Reprvado' (5.5 é maior que 7 caso seja True será Aprovado, caso seja False será Reprovado). O resultado é 'Aprovado'


Exemplo:

var x = 8
